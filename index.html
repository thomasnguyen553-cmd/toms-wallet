Setup Firebase
App cần Firebase để đồng bộ dữ liệu. Làm theo 4 bước này (5 phút, trên điện thoại được):
BƯỚC 1
Vào console.firebase.google.com → đăng nhập Google → "Add project" → đặt tên toms-wallet → tắt Analytics → Create project
BƯỚC 2
Trong project → nhấn icon </> (Web) → đặt tên app → Register app → copy toàn bộ firebaseConfig object
BƯỚC 3
Trong project → Build → Realtime Database → Create database → chọn region us-central1 → chọn Start in test mode → Enable
BƯỚC 4
Mở file index.html → tìm FIREBASE_CONFIG → paste 7 giá trị từ config vào → lưu → upload lên GitHub
💡 Sau khi setup xong, app sẽ tự tạo link duy nhất cho bạn. Chia sẻ link đó để dùng trên mọi thiết bị.
